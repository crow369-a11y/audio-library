# Медиатека

- реализация программы для управления аудиозаписями.

## Функции:
- Просмотр всех треков с сортировкой: исполнитель ↑, год ↓, прослушивания ↓
- Фильтрация по конкретному исполнителю: альбом ↓, название ↑
- Фильтрация по диапазону лет: год ↓, исполнитель ↑

## Использованные технологии:
- Python 3
- Сортировка Хоара (Quicksort)
- CSV-файл как база данных

## Структура проекта:
- audio_library.csv — база данных
- models.py — структура записи
- io_handler.py — чтение данных,обработчик ввода-вывода
- quicksort.py — сортировка
- reports.py — формирование отчётов
- main.py — главное меню
- screenshots — доказательства работы

## Скриншоты:
- главное меню
- обработка ошибок ввода

## Как запустить
1. Скачайте репозиторий
2. Убедитесь, что `audio_library.csv` находится в той же папке, что и `main.py`
3. Запустите: `python3 main.py`
